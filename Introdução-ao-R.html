Operadores Lógicos

TRUE e FALSE (T ou F) são valores lógicos.

O R tem todos os operadores lógicos padrão

2 > 3       # maior que
## [1] FALSE
2 >= 2      # maior ou igual a
## [1] TRUE
1 == 2      # exatamente igual a
## [1] FALSE
2 != 3      # diferente que 
## [1] TRUE

Operadores Lógicos

É possível combinar multiplos operadores lógicos

(2 > 3) | (2 != 4)      # x ou y
## [1] TRUE
(3 > 1) & (2 == 4)      # x e y
## [1] FALSE

Atribuindo Valores

No R, é possível atribuir valores à objetos

object <- value

em que \(object\) é o nome do objeto e \(value\) é o valor atríbuido a ele.

x <- 3    # atribui o valor 3 ao objeto x
y <- 2    # atribui o valor 2 ao objeto y

x * y
## [1] 6

Object Oriented Programming

É um modelo de programação organizado em torno de objetos.

Tudo no R é um objeto.

  • “Objeto” é um termo genérico para descrever alguma coisa no R.

Um objeto é uma estrutura de dados, com atributos específicos.

  • Números, vetores, banco de dados, funções, matrizes, etc.

Cada objeto \(x\) na linguagem R tem uma “classe” associada, que indica o tipo de objeto que \(x\) representa

  • numeric, integer, character, factor, matrix, list, data.frame

A classe de \(x\) indica ao R como interagir com o objeto.

Definindo Funções

Sintaxe básica

function_name <- function(arg_1, arg_2, ...) {
  FunctionStructure
  return()
}

Por exemplo,

caixa = function(M) {
  troco = M - 1.63
  return(troco)
}
caixa(5)
## [1] 3.37

Definindo Funções

Escreva uma função que compute o quadrado e a raiz de um número.

nome <- function(x) {
  quad = x**2
  raiz = sqrt(x)
  return(c(quad, raiz))
}

nome(4)
## [1] 16  2

Regressão Linear

Para regressão linear, a função \(lm\) é uma opção

modelo <- lm(demand ~ price + age, data = DataSet)
